项目简介



本项目使用\*\*动态规划算法\*\*求解D{0-1}背包问题（D{0-1} Knapsack Problem），是经典0-1背包问题的拓展形式。D{0-1}KP中每个项集包含3个物品，第三项具有折扣特性（价值等于前两项之和，重量小于前两项之和），每个项集至多选择一个物品装入背包。问题要求在不超过背包载重的前提下，使装入物品的总价值最大。



本项目提供完整的解决方案，包括数据读取、可视化、排序、求解和结果导出等功能，并支持多数据集处理。



基本功能

文件读取：支持读取标准D{0-1}KP数据文件格式（IDKP系列）

多数据集支持：单个文件可包含多个数据集，支持动态切换

数据可视化：绘制物品重量-价值散点图，支持中文显示

排序功能：按项集第三项的价值重量比进行非递增/递增排序

动态规划求解：

标准动态规划算法（二维数组）

优化动态规划算法（一维数组，节省内存）

结果导出：支持TXT和Excel两种格式导出求解结果

友好界面：命令行交互菜单，操作简单直观



扩展功能

性能优化：一维数组DP算法，降低空间复杂度

多格式导出：同时支持TXT和Excel格式导出

增强可视化：不同项集不同颜色，物品3特殊标注

错误处理：完善的异常捕获和用户提示

数据统计：显示求解时间、选中物品数、总重量等信息
